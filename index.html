    Master Thesis

    大学の修士課程での研究内容です。
    タイトル：
    Blocklyを用いた非手続き型言語のためのプログラミング環境の構築
    http://ymir.eng.kagawa-u.ac.jp/~sano/

    対応言語：
    C / JavaScript / Haskell / Flex / Prolog / Scheme /
    Python
    あらまし：
    学習者が第2, 第3のプログラミング言語を学習するとき, 基礎概念と文法を短時間に学習しなければならない. これは, 学習者にとって大きな負担である. これを解決するために本研究ではブロック方式の初学者用プログラミング環境であるBlocklyを採用し, Blocklyのブロックの動的変形機能を利用して, 手続き型以外の言語に対するプログラミング環境の構築を試みた.
